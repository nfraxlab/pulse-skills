# pulse-skills

Shared skills repository for [Pulse](https://github.com/nfraxlab/pulse). Skills published from the Pulse app are stored here as `SKILL.md` files and automatically indexed by the marketplace.

## Structure

```
skills/
  <skill-id>/
    SKILL.md
```

Each `SKILL.md` follows the standard format:

```markdown
---
name: My Skill
description: What this skill does
---

Instructions and content for the skill.
```

## Publishing

Skills can be published directly from the Pulse macOS app via the skill detail view (menu > Publish to Marketplace).
